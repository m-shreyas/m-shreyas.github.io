## About
Data Analyst portfolio built with vanilla HTML/CSS/JS. Features a Bloomberg-style live data stream, interactive SQL playground, physics-based skill galaxy, and a fully functional CLI terminal.

## Highlights
- 🔴 Live Bloomberg terminal data stream throughout the page
- ⚡ Interactive SQL playground with autocomplete, charts & query history
- 🎮 Physics-based skill galaxy — drag and fling skills
- 💻 Interactive CLI terminal with real commands
- 🌓 Night / Light mode toggle
- 📱 Fully mobile responsive
